Cube Run

Build Date : June 2022
Category : 3D PC Game
Engine : Unity
Language : C#
Github

With the interest of building my first 3-D game, I took on the project of creating Cube Run. This game was built using Unity and C#. The game contains 3-D animations, physics and rigid body systems, as well as player and camera movement scripts that I programmed using C#. This was my first big attempt to step into the game development universe.
