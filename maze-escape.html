Maze Escape

Build Date : August 2022
Category : 3D PC Game
Engine : Unity
Language : C#

Maze Escape as my first attempt to build a game with the aid of Blender and Unity terrain assets. The game utilizes player movement scripts that I programmed using C#, 3-D physics systems, and high quality Unity terrain assets. Maze Escape is a 3-D design project that I created with the simple goal of navigating yourself through a complex maze.
