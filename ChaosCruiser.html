Chaos Cruiser

Build Date : July 2025
Category : Game
Engine : Unity, AirConsole
Language : C#, HTML
Github Play

I contributed to the development of a multiplayer AirConsole game by building and customizing the player controller interface using HTML, CSS, and JavaScript, implementing dynamic player-specific UIs with unique backgrounds, responsive layouts, and custom image-based buttons. On the Unity side, I developed and refined C# gameplay systems including player movement mechanics and win/lose condition logic, ensuring smooth interaction between mobile controllers and in-game events. My work focused on creating scalable, maintainable code while optimizing real-time synchronization across devices, resulting in a seamless integration between the controller interface and Unity’s core gameplay loop.

This game has been built and deployed for the public on itch.io and AirConsole!

The game was also presented at the 2025 Climate Jam competition!
