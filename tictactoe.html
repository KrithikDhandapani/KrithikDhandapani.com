TicTacToe

Build Date : May 2024
Category : Game
Engine : Phaser
Language : JavaScript

TicTacToe is a classic two-player game implemented using JavaScript and the Phaser game framework. This digital adaptation features a scorekeeper that maintains the players' scores across game resets, ensuring a continuous tally of wins for both players. The AI component is designed to play as 'O' against the human player 'X'. The AI intelligently evaluates the board state to either make a winning move or block the player from winning, adding a strategic challenge to the game. When neither condition is met, the AI selects a random empty cell for its move. The game is built in JavaScript, leveraging Phaser's capabilities for dynamic graphics, interactivity, and game logic, providing an engaging and responsive gaming experience.
