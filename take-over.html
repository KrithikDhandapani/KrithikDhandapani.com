Take Over

Build Date : February 2023
Category : Game
Engine : Unity
Language : C#
Github

Being inspired by the most trending games at the time such as Call of Duty, Apex Legends, Valorant, etc​., I wanted to make my own contribution to the first person shooter gaming universe, and so I built Take Over. This game was built using the Unity game engine and C#. It contains fully developed physics systems, FPS player movement scripts and mechanics, and zombies operated by A.I that I programmed using C#.
