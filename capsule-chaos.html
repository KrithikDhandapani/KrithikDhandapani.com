Capsule Chaos : Mobile Edition

Build Date : December 2022
Category : 2-D Mobile Game
Engine : Unity
Language : C#
Github

After creating the 3-D, PC version of Capsule Chaos a few years ago, I decided to create a completely new version of it on mobile. Capsule Chaos : Mobile Edition was created using Unity and C#. The game was my attempt to create a replica of the popular video game Super Smash Bros. Unlike the original version, this version exists on a 2-dimensional plane and features a completely rebuilt physics system suited for 2-D gameplay.
